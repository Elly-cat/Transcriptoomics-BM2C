## inleiding
Reumatoïde artritis is een systematische auto-immunziekte dat voornamelijk de gewrichten van voeten en handen aantast. Alhoewel de exacte oorzaak Reumatoïde artritis onduidelijk is, is het zeker dat omgevingsfactoren en genetischefactoren van invloed zijn op Reumatoïde artritis (Huang et al., 2021).
